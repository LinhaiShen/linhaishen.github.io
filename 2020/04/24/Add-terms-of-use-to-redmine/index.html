Add terms of use to redmine

  1. Install “Privacy & Terms plugin for Redmine” By above codes, you’ve also installed redmine plugin “additionals” which is also very useful.
  2. Create a project to host wiki page of terms where everyone can access
  3. Create wiki page of terms, optionally you can create a wiki page for users who reject the terms
    “the two macros must be implemented in it.”
  4. Config /settings/plugin/redmine_privacy_terms?tab=terms
    URL of the wiki with the terms: wiki/
    URL of the wiki for users who reject the terms: external url or a wiki page (didnot work in my setting)
    tick ‘Enable terms’
  5. Check result in user profile page /users/user-id, and test accept/reject action.
    • Terms of use
      • Show
      • not acccepted yet
  6. Optionally you can re-trigger request users to read and accept terms /settings/plugin/redmine_privacy_terms?tab=tools
    It’s done!
