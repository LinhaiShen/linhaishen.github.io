Synchronize between multiple hexo bases

Posted on 2020-03-08

your.github.io setup

  1. Initial your.github.io with hexo deployment
  2. Create a branch named “hexo” from your.github.io master branch
  3. Set “hexo” as your default branch of your.github.io

    your hexo bases setup

  4. Git clone git@github.com:your/your.github.io.git
  5. Change dir to your cloned “hexo”, install packages for hexo
  6. Make sure deploy branch in _config.yml is master
  7. Write with hexo new post
  8. Deploy to your.github.io master branch with hexo g -d

    Synchronize

  9. git clone for inintial hexo base
  10. git pull for synchronize
  11. git add . after writing
  12. git commit & push to your.github.io “hexo” branch
